Variabelen in Python

Een variabele is een plek in het geheugen waar je een waarde in kan opslaan. Je kan een variabele zien als een doos waar je een waarde in kan stoppen.

In Python kan je een variabele maken door de naam van de variabele te schrijven en daarna een waarde te geven. Je kan de waarde van een variabele veranderen door de naam van de variabele te schrijven en daarna een nieuwe waarde te geven.

Je kan de waarde van een variable de input van een gebruiker maken. Dit doe je door de functie input() te gebruiken.
Je kan de waarde van een variabele printen door de functie print() te gebruiken.

Variabelen maken

              
kosten = 1.0
print(kosten) #Output: 1.0

kosten = 2.0
print(kosten) #Output: 2.0

aantal = input(Aantal: )
print(aantal) #Output invoer

Je kunt ook rekenen met variabelen. Je kan bijvoorbeeld een variabele maken die de prijs van een product is. Als je dan een variabele maakt die de hoeveelheid van het product is, kan je de prijs van het product berekenen.
Dit doe door de variabele totaal te maken en de prijs te vermenigvuldigen met de hoeveelheid.


Reken Operatoren

+ is plus:   75.0 + 4.0 = 79.0   telt waarden bij elkaar op
- is min:   75.0 - 4.0 = 71.0   haalt waarden van elkaar af
* is keer:   75.0 * 4.0 = 300.0   vermenigvuldigt waarden met elkaar
/ is delen:   75.0 / 4.0 = 18.75   deelt waarden door elkaar
% is rest:   75.0 % 4.0 = 3.0   berekent de restwaarde na deling

Rekenen

              
prijs = 2.0
hoeveelheid = 3
totaal = prijs * hoeveelheid
print(totaal) #Output: 6.0

Datatypes in Python

Een datatype is een soort waarde. Er zijn verschillende datatypes. De meest gebruikte datatypes zijn:
Een Integer (int) is een heel getal. Bijvoorbeeld: 1, 2, 3, 4, 5, 6, 7, 8, 9 en 10
Een Float (float) is een kommagetal. Bijvoorbeeld: 1.0, 2.3, 3.64 en 4.234
Een String (str) is een stuk tekst. Bijvoorbeeld: "Hallo", "Dit is een string" en "123"
Een Boolean (bool) is een Ja of Nee. Bijvoorbeeld: True en False
Een List (list) is een lijst met waarden. Bijvoorbeeld: [1, 2, 6, 9]

Gebruik datatypes

              
aantal = 5 #int
kosten = 2.0 #float
productNaam = "Koekje" #str
uitverkocht = False #bool
sterren = [4, 5, 3, 4, 5] #list

Datatypes zijn handig om het verschil aan te geven tussen de tekst "prijs" en de variabele prijs.

Soms moet je de datatype van een variabele veranderen. Bijvoorbeeld als je de waarde van een variabele in een stuk tekst wil noemen. Dan moet het getal een stukje tekst worden.

Je kan de datatype van een variabele opvragen door de functie type() te gebruiken. Je kan de datatype van een variabele veranderen door de datatype te gebruiken als functie. Je moet dan de variabele tussen de haakjes zetten.

Datatype veranderen

              
prijs = 5.0 #float
print(type(prijs)) #Output: float
text = "De prijs: €" + str(prijs)

print(text) 
#Output: De prijs: €5.0

print(type(str(prijs))) 
#Output: str

If-statements in Python

Een if-statement is een stuk code dat alleen wordt uitgevoerd als een voorwaarde waar is. De voorwaarde is een stuk code dat een Boolean waarde geeft. Als de waarde True is, wordt de code uitgevoerd. Als de waarde False is, wordt de code niet uitgevoerd.

Je kan een if-statement maken door het woord if te schrijven en daarna een voorwaarde te zetten. Daarna moet je een dubbele punt schrijven en een tab gebruiken om de code te schrijven die wordt uitgevoerd als de voorwaarde waar is.
Als je de code wil schrijven die wordt uitgevoerd als de voorwaarde niet waar is, moet je na de if-statement else schrijven. Daarna moet je weer een dubbele punt schrijven en een tab gebruiken om de code te schrijven die wordt uitgevoerd als de voorwaarde niet waar is.

If-statement

              
aantal = 5
if aantal > 0:
  print("Je hebt er" + str(aantal) + " gekocht!")
else:
  print("Je moet er minimaal 1 kopen!")

Je kan ook een if-statement maken die meerdere uitgangen heeft. Dit doe je door het woord elif te schrijven en daarna een voorwaarde te zetten. Daarna moet je een dubbele punt schrijven en een tab gebruiken om de code te schrijven die wordt uitgevoerd als de voorwaarde waar is.
Als je de code wil schrijven die wordt uitgevoerd als de voorwaarde niet waar is, kun je ook hierna else schrijven. Daarna moet je weer een dubbele punt schrijven en een tab gebruiken om de code te schrijven die wordt uitgevoerd als bijde voorwaarden niet waar zijn.


Vergelijking en Logische Operatoren

== is gelijk aan: True als links gelijk is aan rechts
> is groter dan: True als links meer is dan rechts
>= is groter of gelijk aan: True als links meer of gelijk is aan rechts
!= is niet gelijk aan: True als links niet gelijk is aan rechts

& of and is en: True als links en rechts True zijn
| of or is of: True als links of rechts True is
! of not is niet: True als niet True is

Elif-statement

              
aantal = 5
if aantal > 0:
  print("Je hebt er" + str(aantal) + " gekocht!")
elif aantal == 0:
  print("Je moet er minimaal 1 kopen!")
else:
  print("Je kan er niet minder dan 0 kopen!")

Loops in Python

Loops zijn handig wanneer je hetzelfde stukje code vaker wil herhalen.


Een van de loops in Python is de while-loop. Deze loop herhaalt iets tot het statement True is.
Dit kan handig zijn wanneer je wilt checken of een gebruiker een geldige waarde heeft ingevoerd. Als de gebruiker een ongeldige waarde heeft ingevoerd, kan je de gebruiker vragen om een nieuwe waarde in te voeren. Dit kan je herhalen tot de gebruiker een geldige waarde heeft ingevoerd.

While-loop

              
aantal = input(Aantal: )
while aantal <= 0:
  print("Je moet er minimaal 1 kopen!")
  aantal = input(Aantal: )

Een andere loop in Python is de for-loop. Deze loop herhaalt iets voor elke waarde in een lijst.
Dit kan handig zijn wanneer je een lijst met waarden hebt en je iets met elke waarde in de lijst wil doen. Bijvoorbeeld als je een lijst met het aantal gegeven sterren hebt en je wil een gemiddelde berekenen.

For-loop

              
sterren = [4, 5, 3, 4, 5]
totaal = 0
for ster in sterren:
  totaal = totaal + ster

#len(lijst) geeft lengte van lijst
print(totaal / len(sterren))
#Output: 4.2
