Object oriented prigramming

💻 Programeren


Object oriented programming is een manier van programmeren waarbij je code opdeelt in objecten. Deze objecten hebben eigenschappen en methodes.
Het is handig om object oriented programming te gebruiken als je een groot project maakt waarin je werkt met veel verschillende dezelfde soort objecten.


In deze vaardigheid leer je hoe je classes maakt en hoe je objecten maakt van deze classes. Ook leer je hoe je objecten kan gebruiken en hoe je objecten kan aanpassen.

Classes in Python

Een class is een blauwdruk voor een object. In een class staan de eigenschappen en methodes van een object.
Je maakt een class door het woord class te schrijven, daarna de naam van de class en te eindigen met een dubbele-punt.
In een class kan je een functie maken die de eigenschappen van een object bepaalt. Deze functie heet de constructor.
Een constructor kun je maken door het woord def te schrijven en daarna __init__ met haakjes erachter.
In de constructor kan je de eigenschappen van een object maken. Dit doe je door het woord self te schrijven, daarna een punt en daarna de naam van de eigenschap.
Je kunt de eigenschap van het object nu als een variabele gebruiken in de class.

Class maken

              
class Persoon:
  def __init__(self):
    self.naam = "Joël"

persoon1 = Persoon()
print(persoon1.naam)
#Output:
#Joël

Je kan aan de constructor ook waarden meegeven. Deze waarden kan je gebruiken om de eigenschappen van het object te bepalen.
Je kan de waarden meegeven door ze tussen de haakjes te schrijven.


Functie en object varabelen

Er is verschil tussen functie variabelen en object variabelen.
De functie variabele word alleen in de functie gebruikt.
In dit voorbeeld is de functie variabele de variabele naam.
De object variabele word gebruikt in de hele class.
In dit voorbeeld is de object variabele de variabele self.naam.

Eigenschappen meegeven

              
class Product:
  def __init__(self, naam, prijs):
    self.naam = naam
    self.prijs = prijs

product1 = Product("Appel", 0.65)
print(product1.naam)
print(product1.prijs)
#Output:
#Appel
#0.65

Wanneer je een object print, krijg je de locatie van het object te zien. Dit is niet handig als je wilt weten waar het object over gaat.
Je kan de functie __str__ maken om te bepalen wat er geprint wordt als je een object print.
Je kan de functie __str__ maken door het woord def te schrijven en daarna __str__ met haakjes erachter.
In de functie __str__ kan je een string returnen. Deze string wordt geprint als je het object print.

__str__ functie

              
class Product:
  def __init__(self, naam, prijs):
    self.naam = naam
    self.prijs = prijs

  def __str__(self):
    return self.naam + ": €" + str(self.prijs)

product1 = Product("Appel", 0.65) 
print(product1)
#Output:
#Appel: €0.65

Class functies in Python

Een class kan ook functies hebben. Deze functies worden methodes genoemd.
Je kan een methode maken door in de class het woord def te schrijven, daarna de naam van de methode en daarna haakjes.
In de methode kan je de eigenschappen van het object gebruiken. Dit doe je door het woord self te schrijven, daarna een punt en daarna de naam van de eigenschap.
Je kan de methode gebruiken door het object te selecteren, daarna een punt en daarna de naam van de methode.

Methode maken

              
class Product:
  def __init__(self, naam, prijs):
    self.naam = naam
    self.prijs = prijs

  def printProductPage(self):
    print("Dit is de productpagina van " + self.naam)
    print("De prijs is €" + str(self.prijs))

product1 = Product("Appel", 0.65)
product1.printProduct()
#Output:
#Dit is de productpagina van Appel
#De prijs is €0.65

Je kan ook een methode maken die een waarde teruggeeft. Dit doe je door het woord return te schrijven en daarna de waarde die je wilt teruggeven.
Je kan de waarde die de methode teruggeeft opslaan in een variabele.
Ook kun je waarden meegeven aan een methode. Dit doe je door de waarden tussen de haakjes te schrijven.
Deze waarde kan je gebruiken in de methode.

Methode met return

              
class Product:
  def __init__(self, naam, prijs):
    self.naam = naam
    self.prijs = prijs

  def totKosten(self, hoeveelheid):  
    return self.prijs * hoeveelheid

item1 = Product("Appel", 0.65)
item2 = Product("Banaan", 0.50)
tot = item1.totKosten(4) + item2.totKosten(2)
print(tot)
#Output:
#3.6

Class Inheritance in Python

Je kan een class maken die de eigenschappen en methodes van een andere class overneemt.
Dit doe je door de naam van de class te schrijven, daarna haakjes en daarna de naam van de class die je wilt overnemen.
Je kan de eigenschappen en methodes van de class die je overneemt gebruiken in de nieuwe class.
Je kan ook nieuwe eigenschappen en methodes maken in de nieuwe class.

In dit voorbeeld hebben we een class Product gemaakt. Deze class heeft de eigenschappen naam en prijs.
Ook heeft deze class de methode printProduct.

Daarna hebben we een class Voedsel gemaakt. Deze class neemt de eigenschappen en methodes van de class Product over.
Ook heeft deze class de eigenschap houdbaarheid en de methode printProduct.

De methode printProduct van de class Voedsel print eerst de methode printProduct van de class Product.
Daarna print deze methode de eigenschap houdbaarheid.

De super() functie wordt gebruikt om de eigenschappen en methodes van de class die je overneemt te gebruiken.

Class overnemen

              
class Product:
  def __init__(self, naam, prijs):
    self.naam = naam
    self.prijs = prijs

  def printProduct(self):
    print("Dit is de productpagina van " + self.naam)
    print("De prijs is €" + str(self.prijs))

class Voedsel(Product):
  def __init__(self, naam, prijs, houdbaarheid):
    super().__init__(self, naam, prijs)
    self.houdbaarheid = houdbaarheid

  def printProduct(self):
    super().printProduct()
    print("De houdbaarheid is " + str(self.houdbaarheid) + " dagen")

product1 = Voedsel("Appel", 0.65, 7)
product1.printProduct()
#Output:
#Dit is de productpagina van Appel
#De prijs is €0.65
#De houdbaarheid is 7 dagen

Libraries in Python

Er zijn ook stukjes code die anderen al hebben geschreven en die je kan gebruiken in je eigen code. Deze stukjes code worden libraries genoemd.
Een library is een verzameling van functies die je kan gebruiken in je code. Je kan een library importeren door het woord import te schrijven en daarna de naam van de library.
Je kan ook een specifieke functie importeren door het woord from te schrijven, daarna de naam van de library en daarna het woord import en de naam van de functie.
Je kan ook een alias geven aan een library of functie door het woord as te schrijven, dan as daarna de naam van de alias.

In dit voorneeld hebben we de library datetime geimporteerd. Deze library wordt gebruikt om te werken met datums en tijden.

Libraries gebruiken

              
import datetime as dt

datum = dt.datetime.now()
print(datum)
#Output:
#huidige datum en tijd bijv:
#2023-10-23 14:24:00.000000

De datetime library brengt een paar nieuwe datatypes met zich mee.
De date datatype wordt gebruikt om een datum op te slaan.
De time datatype wordt gebruikt om een tijd op te slaan.
De datetime datatype wordt gebruikt om een datum en tijd op te slaan.

Je kan een datum maken door het woord date te schrijven en daarna de datum tussen haakjes te schrijven.
Je kan dates ook vergelijken met elkaar. Je kan bijvoorbeeld kijken of een datum voor of na een andere datum is.

Datums vergelijken

              
import datetime as dt

datum1 = dt.date(2023, 10, 23)
datum2 = dt.date(2023, 10, 24)
print(datum2 > datum1)
#Output:
#True

Ook kn je de tijd tussen 2 datums berekenen. Dit kunje doen om een aftelklok te maken voor hoeveel dagen het nog duurt tot een actie is afgeloen bijvoorbeeld.

Tijd tussen 2 datums berekenen

              
import datetime as dt

datum1 = dt.date(2023, 10, 23)
datum2 = dt.date(2023, 10, 24)
tijdVerschil = datum2 - datum1
print(tijdVerschil)
#Output:
#1 day, 0:00:00
