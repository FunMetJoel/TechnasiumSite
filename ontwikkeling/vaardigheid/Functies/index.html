💻 Programeren


Functies is de eerste stap naar het geordend schrijven van code. Met functies kun je code hergebruiken en code opdelen in kleinere stukken.


In deze vaardigheid leer je hoe je functies maakt en gebruikt. Ook leer je een paar standaard functies die je kan gebruiken.

Functies in Python

Een functie is een stuk code dat je kan hergebruiken. Het is handig om een lang stuk code op te delen in meerdere functies, die na elkaar gerunt worden.
Je kan een functie maken door het woord def te schrijven en daarna de naam van de functie te schrijven. Daarna moet je een dubbele punt schrijven en een tab gebruiken om de code te schrijven die wordt uitgevoerd als de functie wordt aangeroepen.


Korte functies maken

Het is good-practice om een functie kort te houden en maar 1 taak te laten doen.


Als je de functie wil gebruiken, moet je de naam van de functie schrijven en daarna haakjes.

Functie maken en gebruiken

              
def printOpening():
  print("Welkom bij de winkel")
  print("Dit kun je kopen:")
  spullen = ["Appel","Peer","Banaan"]
  for product in spullen:
    print(product)
  print("Wat wil je kopen?")

printOpening()
#Output:
#Welkom bij de winkel
#Dit kun je kopen:
#Appel
#Peer
#Banaan
#Wat wil je kopen?

Aan functies kun je ook waarden meegeven. Deze waarden worden parameters genoemd. Je kan de parameters gebruiken in de functie. Als je de functie aanroept, moet je de waarden meegeven.
Bij het maken van de funcite moet je de parameters tussen de haakjes schrijven. Je kan meerdere parameters maken door ze met een komma te scheiden. Je kunt deze variabelen nu alleen binnen de functie gebruiken
Als je de functie aanroept, moet je de waarden tussen de haakjes schrijven.

Variabelen meegeven

              
def printProduct(naam, prijs, uitverkocht):
  print("Koop een " + naam)
  print("Het kost €" + str(prijs))
  if uitverkocht:
    print("Product helaas uitverkocht")

printProduct("Appel", 0.65, False)
#Output:
#Koop een Appel
#Het kost €0.65

printProduct("Peer", 0.75, True)
#Output:
#Koop een Peer
#Het kost €0.75
#Product helaas uitverkocht

Je kan ook een waarde teruggeven met een functie. Dit doe je met het woord return. Je kan de waarde die je teruggeeft ook gebruiken in de code die de functie aanroept.
Je kan de waarde die de functie teruggeeft bijvoorbeeld opslaan in een variabele.

Return

              
def berekenPrijs(aantal, prijs, actie):
  totaal = aantal * prijs
  korting = totaal * actie
  totaal = totaal - korting
  return totaal

totaalPrijs = berekenPrijs(4, 0.65, 0.2)
print(totaalPrijs)
#Output:
#2.08

Standaard functies in Python

Er zijn een aantal standaard functies die je kan gebruiken in Python. Deze functies zijn al gemaakt en je hoeft ze alleen maar aan te roepen. Sommige daarvan heb je al eerder gezien:
Een voorbeeld van zo'n functie is de print() functie. Deze functie print de waarde die je meegeeft in de console.
Een andere functie is de input() functie. Deze functie vraagt de gebruiker om een waarde in te voeren. Deze waarde wordt dan teruggegeven door de functie.
De functie len() zit ook standaard in python en geeft de lengte van een string of lijst terug.
De functie range() maakt een lijst met alle nummers in een range.

Er zijn functies die je kunt gebruiken om te rekenen:
De functie abs() geeft de absolute waarde van een waarde terug.
De functie pow() geeft de macht van een waarde terug.
De functie round() rond een waarde af.

Er zijn functies die helpen met het werken met tekst:
De functie split() splitst een string op een bepaald karakter.
De functie lower() zet een string om naar kleine letters.
De functie upper() zet een string om naar hoofdletters.
De functie strip() verwijdert spaties aan het begin en einde van een string.
De functie replace() vervangt een karakter in een string.

Er zijn functies die helpen met het werken met lijsten:
De functie append() voegt een waarde toe aan een lijst.
De functie remove() verwijdert een waarde uit een lijst.
De functie pop() verwijdert een waarde uit een lijst op een bepaalde index.
De functie sort() sorteert een lijst.
De functie reverse() draait de volgorde van een lijst om.

Standaard functies

              
print(input()) #Output input
print(len("ABC")) #Output: 3
print(range(5)) #Output: [0,1,2,3,4]

print(abs(-5)) #Output: 5
print(pow(2,3)) #Output: 8
print(round(3.14159, 2)) #Output: 3.14

print("Appel Peer Banaan".split(" ")) 
  #Output: ["Appel","Peer","Banaan"]
print("Appel".lower()) #Output: appel
print("Appel".upper()) #Output: APPEL
print(" Appel ".strip()) #Output: Appel
print("Appel".replace("A","E")) #Output: Eppel

print(["Appel","Peer","Banaan"].append("Kiwi")) 
  #Output: ["Appel","Peer","Banaan","Kiwi"]
print(["Appel","Peer","Banaan"].remove("Peer")) 
  #Output: ["Appel","Banaan"]
print(["Appel","Peer","Banaan"].pop(1)) 
  #Output: ["Appel","Banaan"]
print(["Appel","Peer","Banaan"].sort()) 
  #Output: ["Appel","Banaan","Peer"]
print(["Appel","Peer","Banaan"].reverse()) 
  #Output: ["Banaan","Peer","Appel"]

Meerdere bestanden in Python

Als je een groot project maakt, kan het handig zijn om je code op te delen in meerdere bestanden. Dit kan je doen door een bestand te maken met functies en deze functies te importeren in een ander bestand.
Je kan een bestand importeren door het woord import te schrijven en daarna de naam van het bestand.
Je kan ook een specifieke functie importeren door het woord from te schrijven, daarna de naam van het bestand en daarna het woord import en de naam van de functie.
Je kan ook een alias geven aan een bestand of functie door het woord as te schrijven en daarna de naam van de alias.

In dit voorneeld hebben we een bestand met de naam functies.py en een bestand met de naam main.py. In het bestand functies.py staan de functies die we willen gebruiken in het bestand main.py.

functies.py

              
def berekenPrijs(aantal, prijs):
  totaal = aantal * prijs
  return totaal

main.py

              
import functies

totaalPrijs = functies.berekenPrijs(4, 0.65)
print(totaalPrijs)
#Output:
#2.6

Libraries in Python

Er zijn ook stukjes code die anderen al hebben geschreven en die je kan gebruiken in je eigen code. Deze stukjes code worden libraries genoemd.
Een library is een verzameling van functies die je kan gebruiken in je code. Je kan een library importeren door het woord import te schrijven en daarna de naam van de library.
Je kan ook een specifieke functie importeren door het woord from te schrijven, daarna de naam van de library en daarna het woord import en de naam van de functie.
Je kan ook een alias geven aan een library of functie door het woord as te schrijven, dan as daarna de naam van de alias.

In dit voorneeld hebben we de library datetime geimporteerd. Deze library wordt gebruikt om te werken met datums en tijden.

Libraries gebruiken

              
import datetime as dt

datum = dt.datetime.now()
print(datum)
#Output:
#huidige datum en tijd bijv:
#2023-10-23 14:24:00.000000

De datetime library brengt een paar nieuwe datatypes met zich mee.
De date datatype wordt gebruikt om een datum op te slaan.
De time datatype wordt gebruikt om een tijd op te slaan.
De datetime datatype wordt gebruikt om een datum en tijd op te slaan.

Je kan een datum maken door het woord date te schrijven en daarna de datum tussen haakjes te schrijven.
Je kan dates ook vergelijken met elkaar. Je kan bijvoorbeeld kijken of een datum voor of na een andere datum is.

Datums vergelijken

              
import datetime as dt

datum1 = dt.date(2023, 10, 23)
datum2 = dt.date(2023, 10, 24)
print(datum2 > datum1)
#Output:
#True

Ook kn je de tijd tussen 2 datums berekenen. Dit kunje doen om een aftelklok te maken voor hoeveel dagen het nog duurt tot een actie is afgeloen bijvoorbeeld.

Tijd tussen 2 datums berekenen

              
import datetime as dt

datum1 = dt.date(2023, 10, 23)
datum2 = dt.date(2023, 10, 24)
tijdVerschil = datum2 - datum1
print(tijdVerschil)
#Output:
#1 day, 0:00:00
